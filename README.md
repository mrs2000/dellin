Installation
---

The preferred way to install this extension is through 
[composer](http://getcomposer.org/download/).

Either run

```
php composer.phar require --prefer-dist mrssoft/dellin "*"
```

or add

```
"mrssoft/dellin": "*"
```

to the require section of your `composer.json` file.
